## Overview

b64-encodePWA is a compact Base64 encoder that runs in the browser and supports progressive web app installation. It is intended for fast, no-fuss use with HTML and current web browsers, so you can access it without setting up a local environment.

It works well when you want a small web utility that can be opened immediately, installed for later use, and launched again like a native app. The interface stays simple and browser-first so encoding stays quick and easy.
